Penrose Tiling Generator
------------------------

A simple web app that renders a Penrose tiling. (P2/tile and dart variety) The app lets you choose a number of subdivisions/deflations when rendering the tileset. As well you can overlay the results of the previous iteration to observe the deflation process.

Read more about <a href="https://en.wikipedia.org/wiki/Penrose_tiling">Penrose tiling</a>.
